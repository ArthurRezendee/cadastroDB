Cadastro DB
Cadastro DB é um projeto feito apenas por mim, Arthur Rezende, como forma de estudo. Para esse Projeto foram usadas as linguagens PHP, HTML, CSS, SQL e um pouco de JAVASCRIPT. O projeto consiste em um formulário onde, ao cadastrar as informações, elas ficam salvas em um banco de dados e podem ser visualizadas na página "visualizar". Você também pode alterar e excluir os dados que registrou nas abas "Alterar" e "Deletar". Sou um desenvolvedor FullStack em formação e esse projeto é muito importante para mim pois me ajudou muito em desenvolver minhas habilidades!
